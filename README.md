# Deep Sea Hunter

A fast-paced underwater shooter built with Phaser 3.

## Play

Open `index.html` in a browser or run a local server:
```bash
python3 -m http.server 8080
```
Then visit http://localhost:8080

## Controls

- **WASD / Arrow Keys** - Move
- **Space** - Fire
- **P / ESC** - Pause
- **Touch** - Move toward finger, auto-fire

## Gameplay

Survive waves of sea creatures and defeat two epic bosses:

**Enemies:** Jellyfish, Swordfish, Anglerfish, Squid, Sawshark, Mines

**Bosses:**
1. Megalodon (after 25 kills)
2. Kraken (after defeating Megalodon)

**Powerups:**
- **F** - Triple shot
- **S** - Speed boost
- **D** - Shield
- **+** - Health
- **Fire** - Piercing fireballs

## Features

- Parallax scrolling backgrounds
- Progressive difficulty
- High score tracking
- Mobile/touch support
- Background music & sound effects

## Tech

- Phaser 3.70.0
- Vanilla JavaScript (ES6 modules)
- No build step required
